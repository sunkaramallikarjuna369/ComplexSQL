CROSS JOIN returns the Cartesian product of two tables - every row from the first table combined with every row from the second table. If Table A has M rows and Table B has N rows, the result will have M √ó N rows.

üîß HOW to use CROSS JOIN?
CROSS JOIN - All RDBMS Universal 
-- Explicit CROSS JOIN syntax
SELECT 
    c.color_name,
    s.size_name
FROM colors c
CROSS JOIN sizes s;

-- Implicit syntax (comma-separated tables)
SELECT 
    c.color_name,
    s.size_name
FROM colors c, sizes s;

-- Generate all date-hour combinations
SELECT 
    d.date_value,
    h.hour_value
FROM dates d
CROSS JOIN hours h
ORDER BY d.date_value, h.hour_value;

-- Product variants (all color-size combinations)
SELECT 
    p.product_name,
    c.color_name,
    s.size_name,
    CONCAT(p.product_name, ' - ', c.color_name, ' - ', s.size_name) AS variant_name
FROM products p
CROSS JOIN colors c
CROSS JOIN sizes s;

-- Generate number sequence (useful for reports)
WITH numbers AS (
    SELECT 1 AS n UNION ALL SELECT 2 UNION ALL SELECT 3
)
SELECT a.n * 10 + b.n AS number
FROM numbers a
CROSS JOIN numbers b;
