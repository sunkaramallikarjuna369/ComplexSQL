# Introduction to SQL

## Overview
SQL (Structured Query Language) is the standard language for managing and manipulating relational databases.

## Contents
- [concept.html](concept.html) - Interactive visualization with 4W+H explanation
- [examples.sql](examples.sql) - SQL examples for all RDBMS

## Quick Reference
- SQL is declarative - you specify WHAT you want, not HOW to get it
- SQL is case-insensitive for keywords (SELECT = select)
- SQL statements end with semicolons
