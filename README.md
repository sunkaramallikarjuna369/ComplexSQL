# ComplexSQL - Comprehensive SQL Tutorial

A complete SQL learning resource covering all RDBMS concepts with interactive HTML visualizations, sample data, and multi-database syntax (SQL Server, Oracle, PostgreSQL, MySQL).

## Learning Approach: 4W+H Format

Each concept is explained using the **4W+H** methodology:
- **What**: Definition and explanation of the concept
- **Why**: Purpose and benefits of using the concept
- **When**: Scenarios and use cases for applying the concept
- **Where**: Database systems and contexts where it applies
- **How**: Step-by-step implementation with syntax and examples

## How to Use This Repository

1. **Sequential Learning**: Start from `01-SQL-Fundamentals` and progress through each section
2. **Topic-Based Learning**: Jump directly to any concept you want to learn
3. **Interview Preparation**: Focus on `16-Interview-Questions` for practice
4. **Reference**: Use as a quick reference for syntax across different databases
